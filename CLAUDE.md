# Repo notes

## Exam study tracking

When the user reports having trouble with, or asks follow-up/clarifying questions about, a specific exam question from `Complexity/2017-2025 Exams/`, append an entry to the "Issues log" section of the matching notes doc under `Complexity/2017-2025 Exams solutions/notes/<exam name> - Q<range>.md` (create the doc/section if it doesn't exist yet, following the existing format: question number, a one-line summary of the issue/confusion, and how it was resolved).

Currently tracked: `Complexity/2017-2025 Exams solutions/notes/Comp 2022-2 moed C - Q6-Q9.md` (covers Q6-Q9).

## Publishing changes

Whenever a file in this repo is created or updated (e.g. an exam notes doc), commit it and push it to `main` — open a draft PR if not already on `main`, and merge it (marking ready-for-review first if needed) rather than leaving it open for approval. Do this without asking first.
